好的，管综考研数学部分虽然考察的是初等数学知识，但范围广，对公式的熟练掌握是高效解题的关键。以下是备考中必须牢记的核心公式，分为几大模块：

### 一、代数部分 (Algebra)

这部分是考试的重中之重，公式繁多且应用灵活。

1.  **绝对值不等式 (Absolute Value Inequalities)**
    *   `|x| < a  ⇔  -a < x < a`  (a > 0)
    *   `|x| > a  ⇔  x > a 或 x < -a` (a > 0)
    *   三角不等式：`|a| - |b| ≤ |a ± b| ≤ |a| + |b|`

2.  **均值不等式 (AM-GM Inequality)**
    *   对于两个正数a, b：`a + b ≥ 2√ab`，当且仅当 `a = b` 时取等号。
    *   对于三个正数a, b, c：`a + b + c ≥ 3∛abc`，当且仅当 `a = b = c` 时取等号。
    *   **核心应用**：求“和为定值，积有最大值”或“积为定值，和有最小值”的问题。

3.  **一元二次方程与函数 (Quadratic Equations and Functions)**
    *   **根的判别式**: `Δ = b² - 4ac`
        *   `Δ > 0` ⇒ 两个不相等的实数根
        *   `Δ = 0` ⇒ 两个相等的实数根
        *   `Δ < 0` ⇒ 没有实数根
    *   **求根公式**: `x = (-b ± √Δ) / 2a`
    *   **韦达定理 (Vieta's Formulas)**:
        *   两根之和: `x₁ + x₂ = -b/a`
        *   两根之积: `x₁ * x₂ = c/a`
    *   **顶点坐标 (Vertex Formula)**: `(-b/2a, (4ac - b²)/4a)`
    *   **两点式/截距式** 等函数表达形式也需掌握。

4.  **数列 (Sequences)**
    *   **等差数列 (Arithmetic Sequence)**:
        *   通项公式: `aₙ = a₁ + (n-1)d`
        *   前n项和: `Sₙ = n(a₁ + aₙ)/2 = na₁ + n(n-1)d/2`
        *   性质: `aₘ + aₙ = aₚ + aₖ` (若 m+n = p+k)
    *   **等比数列 (Geometric Sequence)**:
        *   通项公式: `aₙ = a₁ * q^(n-1)`
        *   前n项和: `Sₙ = a₁(1 - qⁿ) / (1 - q)` (q ≠ 1)
        *   性质: `aₘ * aₙ = aₚ * aₖ` (若 m+n = p+k)

### 二、算术与应用题 (Arithmetic and Word Problems)

这部分公式相对简单，但需要深刻理解其含义。

1.  **比例与百分比 (Ratio and Percentage)**
    *   增长率: `增长率 = (增长量 / 原始量) × 100%`
    *   `a比b多p%`  ⇔ `a = b(1 + p%)`
    *   `a比b少p%`  ⇔ `a = b(1 - p%)`

2.  **行程问题 (Travel Problems)**
    *   `路程 = 速度 × 时间`
    *   相遇问题: `相遇路程 = 速度和 × 相遇时间`
    *   追及问题: `追及路程 = 速度差 × 追及时间`

3.  **工程问题 (Work Problems)**
    *   `工作总量 = 工作效率 × 工作时间`
    *   多人合作: `效率和 = 各自效率之和`

4.  **浓度问题 (Mixture Problems)**
    *   `溶质质量 = 溶液质量 × 浓度`
    *   `混合后浓度 = (溶质总质量 / 溶液总质量) × 100%`

### 三、几何部分 (Geometry)

几何部分需要记忆平面图形和立体图形的周长、面积、体积公式。

1.  **平面几何 (Plane Geometry)**
    *   三角形面积: `S = ½ * 底 * 高 = ½ab * sinC`
    *   圆形周长与面积: `C = 2πr`, `S = πr²`
    *   扇形面积: `S = n/360 * πr² = ½lr` (l为弧长)

2.  **解析几何 (Analytic Geometry)**
    *   两点间距离: `d = √[(x₁-x₂)² + (y₁-y₂)²]`
    *   中点坐标: `((x₁+x₂)/2, (y₁+y₂)/2)`
    *   直线方程:
        *   点斜式: `y - y₀ = k(x - x₀)`
        *   斜截式: `y = kx + b`
        *   一般式: `Ax + By + C = 0`
    *   点到直线的距离: `d = |Ax₀ + By₀ + C| / √(A² + B²)`
    *   圆的标准方程: `(x-a)² + (y-b)² = r²`

3.  **立体几何 (Solid Geometry)**
    *   长方体: 表面积 `S = 2(ab+bc+ac)`, 体积 `V = abc`
    *   圆柱体: 表面积 `S = 2πr² + 2πrh`, 体积 `V = πr²h`
    *   圆锥体: 体积 `V = ⅓πr²h`
    *   球体: 表面积 `S = 4πR²`, 体积 `V = ⁴⁄₃πR³`

### 四、数据分析、概率与计数 (Data Analysis, Probability, and Counting)

1.  **计数原理 (Counting Principles)**
    *   排列 (Permutation): `P(n, m) = n! / (n-m)!` (从n个不同元素中取出m个进行排序)
    *   组合 (Combination): `C(n, m) = n! / [m!(n-m)!]` (从n个不同元素中取出m个不考虑顺序)
    *   常用性质: `C(n, m) = C(n, n-m)`

2.  **概率 (Probability)**
    *   **古典概型**: `P(A) = A包含的基本事件数 / 总基本事件数`
    *   **加法公式**: `P(A+B) = P(A) + P(B) - P(AB)` (若A, B互斥，则`P(A+B) = P(A) + P(B)`)
    *   **条件概率**: `P(B|A) = P(AB) / P(A)`
    *   **乘法公式**: `P(AB) = P(A) * P(B|A)` (若A, B相互独立，则`P(AB) = P(A) * P(B)`)

3.  **统计 (Statistics)**
    *   平均数 (Mean): `x̄ = (x₁ + x₂ + ... + xₙ) / n`
    *   方差 (Variance): `S² = ¹⁄ₙ * Σ(xᵢ - x̄)²`
    *   标准差 (Standard Deviation): `S = √S²`

**备考建议：**
*   **理解重于死记**：理解公式的推导过程和适用场景比单纯背诵更重要。
*   **分类记忆**：将公式按模块分类，建立知识体系框架。
*   **多练多用**：通过大量真题和模拟题的练习，将公式内化为自己的解题工具。
