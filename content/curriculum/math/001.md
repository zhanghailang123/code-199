**重要提示（避坑第一条）：**
在管综数学大纲中，“曲线”**只考圆和直线**。椭圆、双曲线、抛物线这些“圆锥曲线”**完全不考**！千万不要去复习那些，会浪费大量时间。

这份教程分为三个部分：**核心公式库**、**必考三大模型**、**秒杀技巧**。

### 第一部分：核心公式库（必须背写在A4纸上）

解析几何的本质是“数形结合”。一定要记住：**能用几何法，绝不用代数法（联立方程）**。

#### 1. 距离公式（一切题目的基石）
*   **两点间距离：** $\sqrt{(x_1-x_2)^2 + (y_1-y_2)^2}$
*   **点到直线距离（五星级公式）：**
    点 $P(x_0, y_0)$ 到直线 $Ax + By + C = 0$ 的距离 $d$：
    $$d = \frac{|Ax_0 + By_0 + C|}{\sqrt{A^2 + B^2}}$$
    *注：这个公式考频最高，算错必定完蛋。*

#### 2. 圆的方程
*   **标准方程：** $(x-a)^2 + (y-b)^2 = r^2$
    *   圆心 $(a, b)$，半径 $r$。
*   **一般方程：** $x^2 + y^2 + Dx + Ey + F = 0$
    *   **怎么处理？** 看到一般方程，第一反应必须是**配方**，化为标准方程找圆心和半径。

#### 3. 直线与圆的位置关系（判断标准）
不要用 $\Delta$（判别式），计算量太大！请用 **圆心到直线的距离 $d$ 与半径 $r$ 比较**：
*   $d < r \rightarrow$ **相交**（有两个交点，形成弦）
*   $d = r \rightarrow$ **相切**（有一个交点，切线）
*   $d > r \rightarrow$ **相离**（无交点）

### 第二部分：必考三大模型（解题套路）

考试中90%的题目都逃不出这三个模型。

#### 模型一：求弦长（垂径定理三角形）—— 考频 No.1
题目通常给一条直线和一个圆相交，问你截得的弦长是多少。

*   **几何解法（勾股定理）：**
    画一个图：圆心 $O$，弦的中点 $M$，弦的一个端点 $A$。
    这就构成了一个直角三角形 $\triangle OMA$。
    *   斜边 = 半径 $r$
    *   直角边1 = 圆心到直线的距离 $d$
    *   直角边2 = 半弦长 $\frac{L}{2}$
*   **万能公式：**
    $$r^2 = d^2 + (\frac{L}{2})^2 \Rightarrow L = 2\sqrt{r^2 - d^2}$$
*   **做题步骤：**
    1.  找出圆心和半径 $r$。
    2.  用点到直线距离公式算出 $d$。
    3.  代入公式求 $L$。

#### 模型二：圆与圆的位置关系（公切线问题）
题目给你两个圆，问有几条公切线，或者两圆的最短距离。

*   **核心逻辑：** 只看**圆心距 $d$** 与 **两半径之和/差** 的关系。
    设两圆半径分别为 $R, r$ (大圆和小圆)，圆心距离为 $d$。
    *   $d > R+r \rightarrow$ **外离**（4条公切线，距离最远）
    *   $d = R+r \rightarrow$ **外切**（3条公切线）
    *   $R-r < d < R+r \rightarrow$ **相交**（2条公切线）
    *   $d = R-r \rightarrow$ **内切**（1条公切线）
    *   $d < R-r \rightarrow$ **内含**（0条公切线）

#### 模型三：对称问题（关于直线对称）
题目问点关于直线对称，或者圆关于直线对称。

*   **口诀：** 圆关于直线对称，其实就是**圆心关于直线对称**（半径不变）。
*   **核心算法（求点 $P(x_0, y_0)$ 关于直线 $Ax+By+C=0$ 的对称点 $P'$）：**
    不需要背复杂的公式，记住两个性质列方程组：
    1.  **垂直：** $PP'$ 的斜率 $\times$ 直线斜率 = $-1$。
    2.  **平分：** $PP'$ 的中点在直线上（把中点坐标代入直线方程）。

### 第三部分：高分最值问题（北邮冲刺必看）

这部分通常是选择题最后两道，考的是**“几何意义”**。

#### 1. “距离”的最值
看到式子 $\sqrt{(x-a)^2 + (y-b)^2}$，不要去算平方，要立刻反应过来：**这是点 $(x,y)$ 到点 $(a,b)$ 的距离！**

*   **经典考题：** 圆上一点 $P$ 到直线的最短/最长距离。
    *   解法：先算圆心到直线的距离 $d$。
    *   最短距离 = $d - r$
    *   最长距离 = $d + r$

#### 2. “斜率”的最值
看到式子 $\frac{y-b}{x-a}$，要立刻反应过来：**这是点 $(x,y)$ 与点 $(a,b)$ 连线的斜率！**

*   **经典考题：** 求 $\frac{y}{x+2}$ 的最大值（其中 $x,y$ 在圆上）。
    *   解法：看作是圆上的点到 $(-2, 0)$ 的连线斜率。
    *   **极值点：** 连线与圆**相切**的时候，斜率最大或最小。
    *   操作：设斜率为 $k$，写出直线方程，利用 $d=r$ 求出 $k$。

#### 3. “截距”的最值
看到式子 $ax + by$，要立刻反应过来：**这是直线 $ax+by=z$ 的截距相关量！**

*   解法：这是一组平行线。当直线与圆**相切**时，取到最大值或最小值。

### 总结一张小抄（做题时放在手边）

| 看到什么 | 想到什么 | 公式/方法 |
| :--- | :--- | :--- |
| **直线与圆相交/求弦长** | **直角三角形** | $r^2 = d^2 + (L/2)^2$ |
| **直线与圆相切** | **距离等于半径** | $d = r$ |
| **圆与圆外切** | **圆心距=半径和** | $d = R+r$ |
| $\sqrt{(x-a)^2+(y-b)^2}$ | **两点间距离** | 连线找圆心 |
| $\frac{y-b}{x-a}$ | **斜率** | 连线看切线 |
| $Ax+By$ | **平行线移动** | 找切点 |

把这个逻辑吃透，解析几何这部分（约15-18分）你就能稳拿了。
