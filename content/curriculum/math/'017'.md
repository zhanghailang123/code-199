# 017 韦达定理

### 第一部分：核心定义（基础地基）

1.  **一元二次方程标准式**:
    任何一个关于 $x$ 的一元二次方程，都可以化为如下标准形式：
    $ax^2 + bx + c = 0 \quad (a \neq 0)$
    其中，$a$ 为二次项系数，$b$ 为一次项系数，$c$ 为常数项。**特别注意：$a \neq 0$ 是方程成立的绝对前提。**

2.  **根的判别式 (Delta)**:
    判别式是判断方程实数根个数的“照妖镜”。
    $\Delta = b^2 - 4ac$
    *   $\Delta > 0 \iff$ 方程有两个**不相等**的实数根。
    *   $\Delta = 0 \iff$ 方程有两个**相等**的实数根。
    *   $\Delta < 0 \iff$ 方程**没有**实数根。
    *   *推论：* $\Delta \ge 0 \iff$ 方程**有实数根**（包含两个相等或不等）。这是解题中极其重要的隐藏条件。

3.  **韦达定理 (Vieta's Formulas)**:
    当方程 $ax^2 + bx + c = 0$ 有实数根 $x_1, x_2$ 时（即 $\Delta \ge 0$），根与系数之间存在如下关系：
    *   **两根之和**: $x_1 + x_2 = -\frac{b}{a}$
    *   **两根之积**: $x_1 \cdot x_2 = \frac{c}{a}$

4.  **根的倒数和与平方和（常用推论）**:
    这些是基于韦达定理的高频变形，必须熟记。
    *   **倒数和**: $\frac{1}{x_1} + \frac{1}{x_2} = \frac{x_1+x_2}{x_1x_2} = \frac{-b/a}{c/a} = -\frac{b}{c}$
    *   **平方和**: $x_1^2 + x_2^2 = (x_1+x_2)^2 - 2x_1x_2 = (-\frac{b}{a})^2 - 2(\frac{c}{a}) = \frac{b^2-2ac}{a^2}$
    *   **差的平方**: $(x_1-x_2)^2 = (x_1+x_2)^2 - 4x_1x_2 = \frac{b^2-4ac}{a^2} = \frac{\Delta}{a^2}$

### 第二部分：核心考点（考试套路）

#### 考点 1：对称多项式求值

*   **原理：** 利用韦达定理，将关于两根 $x_1, x_2$ 的对称表达式（即交换 $x_1, x_2$ 后表达式不变），全部转化为用 $x_1+x_2$ 和 $x_1x_2$ 来表示，进而代入系数值求解。
*   **公式/结论：**
    *   $x_1^2 + x_2^2 = (x_1+x_2)^2 - 2x_1x_2$
    *   $|x_1 - x_2| = \sqrt{(x_1-x_2)^2} = \sqrt{(x_1+x_2)^2 - 4x_1x_2} = \frac{\sqrt{\Delta}}{|a|}$
    *   $\frac{x_2}{x_1} + \frac{x_1}{x_2} = \frac{x_1^2+x_2^2}{x_1x_2}$
*   **应用场景：** 题目直接给出方程，要求计算含 $x_1, x_2$ 的复杂代数式的值。

#### 考点 2：构造新方程与参数反求

*   **原理：** 若已知新方程的两根与原方程两根的关系，可先用韦达定理求出新方程的两根之和与两根之积，再利用“设根法”构造出新方程。
*   **公式/结论：** 若已知两根为 $m, n$，则对应的一元二次方程可设为 $x^2 - (m+n)x + mn = 0$。**注意：** 二次项系数默认为1，若题目有特殊要求，需等比例放大。
*   **应用场景：** 题目给出方程 $A$ 的两根 $x_1, x_2$，要求构造一个新方程 $B$，其两根为 $x_1+k, x_2+k$ 或 $\frac{1}{x_1}, \frac{1}{x_2}$ 等。或反过来，告知两根满足某种关系，求方程中的待定参数。

### 第三部分：真题逻辑演练（文字解析）

**【例题 1】** (改编自真题)
已知关于 $x$ 的方程 $x^2 - (k+1)x + 2k = 0$ 的两根之差的平方为 9，求参数 $k$ 的值。

*   **文字解析：**
    1.  **识别题型：** 题目给出了关于两根关系的具体数值（差的平方），要求反推方程中的参数，是典型的韦达定理应用。
    2.  **关联考点：** 两根之差的平方 $(x_1-x_2)^2$ 可以通过韦达定理的核心结论 $(x_1-x_2)^2 = (x_1+x_2)^2 - 4x_1x_2$ 来求解。
    3.  **提取信息并计算：**
        *   根据韦达定理，从方程 $x^2 - (k+1)x + 2k = 0$ 中提取两根之和与积：
            *   $x_1 + x_2 = - \frac{-(k+1)}{1} = k+1$
            *   $x_1 \cdot x_2 = \frac{2k}{1} = 2k$
        *   代入公式：$(x_1-x_2)^2 = (k+1)^2 - 4(2k) = 9$
        *   展开并化简：$k^2 + 2k + 1 - 8k = 9 \implies k^2 - 6k - 8 = 0$
    4.  **求解参数：**
        *   这是一个关于 $k$ 的一元二次方程，解得 $k = \frac{6 \pm \sqrt{36 - 4(1)(-8)}}{2} = \frac{6 \pm \sqrt{68}}{2} = 3 \pm \sqrt{17}$。
    5.  **（关键）验根：** 使用韦达定理的前提是方程有实根，即 $\Delta \ge 0$。
        *   原方程的判别式 $\Delta = [-(k+1)]^2 - 4(1)(2k) = k^2 + 2k + 1 - 8k = k^2 - 6k + 1$。
        *   由于 $(x_1-x_2)^2 = \frac{\Delta}{a^2}$，题目已知 $(x_1-x_2)^2 = 9 > 0$，所以 $\Delta > 0$ 必然成立，无需额外检验。
    6.  **结论：** 参数 $k$ 的值为 $3 + \sqrt{17}$ 或 $3 - \sqrt{17}$。

**【例题 2】** (改编自真题)
已知方程 $5x^2 + mx - 2 = 0$ 的两个实数根为 $x_1, x_2$，若 $\frac{1}{x_1} + \frac{1}{x_2} = 3$，求以 $x_1^2, x_2^2$ 为两根的新方程。

*   **文字解析：**
    1.  **分步拆解：** 任务分为两步。第一步，利用已知条件求出参数 $m$；第二步，求出新方程的两根之和与积，构造新方程。
    2.  **第一步：求参数 m**
        *   $\frac{1}{x_1} + \frac{1}{x_2}$ 是典型的韦达定理推论。
        *   由原方程 $5x^2 + mx - 2 = 0$ 可得：$x_1+x_2 = -\frac{m}{5}$，$x_1x_2 = -\frac{2}{5}$。
        *   $\frac{1}{x_1} + \frac{1}{x_2} = \frac{x_1+x_2}{x_1x_2} = \frac{-m/5}{-2/5} = \frac{m}{2}$。
        *   根据题意 $\frac{m}{2} = 3$，解得 $m=6$。
    3.  **第二步：构造新方程**
        *   新方程的两根是 $y_1 = x_1^2$ 和 $y_2 = x_2^2$。我们需要计算 $y_1+y_2$ 和 $y_1y_2$。
        *   **新两根之和：** $y_1+y_2 = x_1^2 + x_2^2 = (x_1+x_2)^2 - 2x_1x_2$。
            *   将 $m=6$ 代入原方程关系式：$x_1+x_2 = -\frac{6}{5}$，$x_1x_2 = -\frac{2}{5}$。
            *   $y_1+y_2 = (-\frac{6}{5})^2 - 2(-\frac{2}{5}) = \frac{36}{25} + \frac{4}{5} = \frac{36+20}{25} = \frac{56}{25}$。
        *   **新两根之积：** $y_1y_2 = x_1^2 \cdot x_2^2 = (x_1x_2)^2 = (-\frac{2}{5})^2 = \frac{4}{25}$。
    4.  **写出方程：** 设新方程为 $y^2 - (y_1+y_2)y + y_1y_2 = 0$。
        *   代入计算结果：$y^2 - \frac{56}{25}y + \frac{4}{25} = 0$。
    5.  **结论（化简）：** 方程两边同乘以 25，得到最终形式：$25y^2 - 56y + 4 = 0$。

### 第四部分：避坑指南（考试心理）

1.  **【天坑一】无脑套公式，不验判别式 $\Delta$**
    *   **错误表现：** 看到题目含参数，立刻使用韦达定理，解出参数后直接作为答案，完全忘记验证该参数是否能保证原方程有实数根。
    *   **防坑口诀：** “韦达前提 $\Delta \ge 0$，解完参数回头瞅，代入 $\Delta$ 辨真伪，小于零者皆舍走。”

2.  **【天坑二】忽略二次项系数 $a \neq 0$**
    *   **错误表现：** 当二次项系数含有参数时（如 $(k-1)x^2+...=0$），没有讨论 $k=1$ 的情况，直接当作一元二次方程处理。
    *   **防坑口诀：** “二次项，有参数，先令它为零，特殊情况要算清。”

3.  **【天坑三】根的实际意义限制**
    *   **错误表现：** 在应用题中，如方程的根代表线段长度、价格等，解出参数后，未验证根本身是否符合实际意义（如长度必须为正数）。
    *   **防坑口诀：** “若是应用题，两根有姓名，长度不能负，价格要看清。”

### 第五部分：考情分析（情报局）

*   **难度星级：** ⭐⭐ (基础知识不难，但题目灵活多变)
*   **考频指数：** ⭐⭐⭐⭐⭐ (几乎是每年必考或作为综合题的一部分出现)
*   **命题趋势：**
    *   **纯粹计算题减少：** 直接让你求 $x_1^2+x_2^2$ 这种“送分题”越来越少。
    *   **与函数、不等式结合：** 常常作为二次函数 $y=ax^2+bx+c$ 与x轴交点问题出现，或结合根的分布（如两根均大于1）来考察，综合性增强。
    *   **隐藏条件增多：** 题目不会直接说“请用韦达定理”，而是通过“两根之比”、“一个根是另一个根的k倍”等形式给出，需要考生主动转化。

### 第七部分：针对 MEM/MBA 的复习建议

1.  **公式求“精”，不做“记忆黑洞”：**
    在职考生时间宝贵，记忆力可能不如学生时代。不要死记硬背所有推论公式。牢牢记住最核心的“两根之和”与“两根之积”，现场推导平方和、倒数和等，既能加深理解，又能减少记忆负担，考场上反而更稳。

2.  **题型归“类”，不做“无头苍蝇”：**
    韦达定理的题型非常固定。将做过的真题和模拟题进行分类，整理出“求值类”、“求参数类”、“构造方程类”、“结合图像类”等几个模板。每个模板掌握一道母题的完整解题逻辑，就能举一反三，以不变应万变。

3.  **真题为“王”，不做“偏题怪题”：**
    管理类联考数学注重基础和应用，不会在韦达定理上出过于刁钻的计算。把近 5-10 年的真题彻底吃透，尤其是那些将韦达定理作为解题步骤之一的综合题。通过真题反推命题人思路，比刷一百道偏难怪题的效率高得多。
