# 017 韦达定理

### 第一部分：核心定义（基础地基）

韦达定理（Vieta's formulas）揭示了一元n次方程中根与系数之间的内在关系，是代数领域的核心工具。在管理类联考中，考察重点集中在一元二次方程。

*   **一元二次方程的标准形式：**
    $ax^2 + bx + c = 0$  (*其中 a ≠ 0*)

*   **根的判别式：**
    $\Delta = b^2 - 4ac$
    *   $\Delta > 0$: 方程有两个不相等的实数根。
    *   $\Delta = 0$: 方程有两个相等的实数根。
    *   $\Delta < 0$: 方程没有实数根。
    **注意：** 使用韦达定理的前提是方程有实数根，即 $\Delta \ge 0$。

*   **韦达定理（二次）：**
    若 $x_1$ 和 $x_2$ 是方程 $ax^2 + bx + c = 0$ 的两个根，则：
    *   两根之和： $x_1 + x_2 = -\frac{b}{a}$
    *   两根之积： $x_1 x_2 = \frac{c}{a}$

*   **常用变形与恒等式：**
    *   平方和： $x_1^2 + x_2^2 = (x_1 + x_2)^2 - 2x_1x_2$
    *   倒数和： $\frac{1}{x_1} + \frac{1}{x_2} = \frac{x_1 + x_2}{x_1x_2}$
    *   两根之差的平方： $(x_1 - x_2)^2 = (x_1 + x_2)^2 - 4x_1x_2$
    *   绝对值之差： $|x_1 - x_2| = \frac{\sqrt{\Delta}}{|a|}$
    *   立方和： $x_1^3 + x_2^3 = (x_1 + x_2)(x_1^2 - x_1x_2 + x_2^2) = (x_1+x_2)^3 - 3x_1x_2(x_1+x_2)$

### 第二部分：核心考点（考试套路）

#### 考点一：对称式求值

*   **原理：** 利用韦达定理求出两根之和 ($x_1+x_2$) 与两根之积 ($x_1x_2$)，再通过恒等变形，将目标式子用这两个基本对称式表达出来并代入求值。
*   **公式/结论：** 核心就是将一切关于根的对称式（交换 $x_1, x_2$ 后表达式不变）都转化为 $x_1+x_2$ 和 $x_1x_2$ 的组合。
*   **应用场景：** 题目直接给出方程，要求计算 $x_1^2+x_2^2$, $\frac{x_2}{x_1}+\frac{x_1}{x_2}$ 等对称表达式的值。

#### 考点二：逆向构造方程

*   **原理：** 如果知道两个数 $m, n$ 的和与积，可以迅速构造出以这两个数为根的一元二次方程。
*   **公式/结论：** 以 $m, n$ 为根的一元二次方程（二次项系数为1）是：$x^2 - (m+n)x + mn = 0$。
*   **应用场景：** 题目已知某方程的两根 $x_1, x_2$，要求构造一个新方程，其根为 $x_1+k, x_2+k$ 或 $\frac{1}{x_1}, \frac{1}{x_2}$ 等。

#### 考点三：结合根的性质判断参数范围

*   **原理：** 将题目中关于根的描述（如“两根均为正数”、“一正一负且负根绝对值更大”）转化为关于 $x_1+x_2$ 和 $x_1x_2$ 的不等式，结合判别式 $\Delta \ge 0$ 求解参数的取值范围。
*   **结论：**
    *   两根均为正数 $\iff \Delta \ge 0, x_1+x_2>0, x_1x_2>0$
    *   两根均为负数 $\iff \Delta \ge 0, x_1+x_2<0, x_1x_2>0$
    *   一正一负 $\iff x_1x_2 < 0$ (此时 $\Delta$ 必大于0)
*   **应用场景：** 方程系数中含有未知参数，题目对根的正负、大小关系等做出限定，要求解该参数的范围。

### 第三部分：真题逻辑演练（文字解析）

**【例题 1】** (改编自真题) 已知关于 $x$ 的方程 $x^2 - (k+2)x + 2k = 0$ 的两根之差的平方为 9，则实数 $k$ 的值为多少？

*   **文字解析：**
    1.  **识别题型与工具：** 题目涉及“两根之差”，这是典型的韦达定理应用题。我们的目标是建立一个关于 $k$ 的方程。
    2.  **应用韦达定理：** 设方程两根为 $x_1, x_2$。根据韦达定理，我们可以得到：
        *   $x_1 + x_2 = - \frac{-(k+2)}{1} = k+2$
        *   $x_1 x_2 = \frac{2k}{1} = 2k$
    3.  **连接已知与未知：** 题目给出的条件是“两根之差的平方为 9”，即 $(x_1 - x_2)^2 = 9$。我们需要将这个式子用 $x_1+x_2$ 和 $x_1x_2$ 来表示。利用恒等式 $(x_1 - x_2)^2 = (x_1 + x_2)^2 - 4x_1x_2$。
    4.  **建立并求解方程：** 将韦达定理的结果代入恒等式中：
        *   $(k+2)^2 - 4(2k) = 9$
        *   $k^2 + 4k + 4 - 8k = 9$
        *   $k^2 - 4k - 5 = 0$
        *   $(k-5)(k+1) = 0$
        *   解得 $k=5$ 或 $k=-1$。
    5.  **检验判别式（关键步骤）：** 别忘了韦达定理的前提是方程有实根 ($\Delta \ge 0$)。
        *   $\Delta = [-(k+2)]^2 - 4(1)(2k) = k^2 + 4k + 4 - 8k = k^2 - 4k + 4 = (k-2)^2$
        *   因为 $(k-2)^2 \ge 0$ 恒成立，所以对于任意实数 $k$，方程都有实数根。因此，我们求出的两个 $k$ 值都有效。
    6.  **结论：** 实数 $k$ 的值为 5 或 -1。

**【例题 2】** (改编自真题) 若方程 $x^2 + 5x + 2 = 0$ 的两根为 $\alpha, \beta$，求以 $\frac{1}{\alpha}+1$ 和 $\frac{1}{\beta}+1$ 为两根的新方程。

*   **文字解析：**
    1.  **识别题型与工具：** 这是考查“逆向构造方程”的典型题目。我们需要求出新方程两根的和与积。
    2.  **获取原始信息：** 对于原方程 $x^2 + 5x + 2 = 0$，由韦达定理可知：
        *   $\alpha + \beta = -5$
        *   $\alpha \beta = 2$
    3.  **计算新根之和：** 设新方程的两根为 $y_1 = \frac{1}{\alpha}+1$ 和 $y_2 = \frac{1}{\beta}+1$。
        *   $y_1 + y_2 = (\frac{1}{\alpha}+1) + (\frac{1}{\beta}+1) = \frac{\alpha+\beta}{\alpha\beta} + 2$
        *   代入原始信息：$y_1 + y_2 = \frac{-5}{2} + 2 = -\frac{1}{2}$
    4.  **计算新根之积：**
        *   $y_1 y_2 = (\frac{1}{\alpha}+1)(\frac{1}{\beta}+1) = \frac{1}{\alpha\beta} + \frac{1}{\alpha} + \frac{1}{\beta} + 1 = \frac{1}{\alpha\beta} + \frac{\alpha+\beta}{\alpha\beta} + 1$
        *   代入原始信息：$y_1 y_2 = \frac{1}{2} + \frac{-5}{2} + 1 = -2 + 1 = -1$
    5.  **构造新方程：** 利用公式 $y^2 - (y_1+y_2)y + y_1y_2 = 0$。
        *   $y^2 - (-\frac{1}{2})y + (-1) = 0$
        *   $y^2 + \frac{1}{2}y - 1 = 0$
    6.  **结论（标准化）：** 为了美观，通常将系数化为整数。方程两边同乘以 2，得到 $2y^2 + y - 2 = 0$。

### 第四部分：避坑指南（考试心理）

1.  **致命错误：** 拿到题目直接用韦达定理，算完就选答案，**完全忘记检验判别式 $\Delta \ge 0$**。
    *   **防坑口诀：** “韦达之前，先判生死！”——先看 $\Delta$ 定根的有无，再谈根的关系。

2.  **计算错误：** 在处理 $x_1+x_2 = -\frac{b}{a}$ 时，经常忘记负号，尤其当 $b$ 本身是负数时。
    *   **防坑口诀：** “两根之和，总与 b 反。”——心里默念一遍，符号就不会错。

3.  **概念混淆：** 对根的性质与韦达定理表达式的对应关系不熟，例如，无法迅速将“两根互为倒数”转化为 $x_1x_2 = 1$。
    *   **记忆技巧：**
        *   互为相反数 $\implies$ 和为 0 $\implies x_1+x_2 = 0 \implies b=0$
        *   互为倒数 $\implies$ 积为 1 $\implies x_1x_2 = 1 \implies c=a$

### 第七部分：针对 MEM/MBA 的复习建议

1.  **回归基础，不做难题：** 对于在职考生，时间宝贵，精力有限。韦达定理这部分，务必将核心公式和常用变形（如平方和、差的平方）背得滚瓜烂熟。不要去钻研偏、难、怪题，把基础分稳稳拿到手才是王道。

2.  **强化“翻译”能力：** 备考的重点应放在识别题干的文字信息，并将其“翻译”成数学语言上。例如，看到“一个根是另一个根的两倍”，就要立刻反应出 $x_1=2x_2$，然后结合 $x_1+x_2$ 和 $x_1x_2$ 求解。这种“翻译”练习比单纯刷题更高效。

3.  **建立“必检项”清单：** 针对容易出错的点，建立一个自己的检查清单。例如，做完韦达定理的题，强制自己检查：① $\Delta \ge 0$ 考虑了吗？ ② $-b/a$ 的负号没错吧？ ③ 题目有没有隐含条件（如三角形两边长必须为正）？通过刻意练习，将易错点内化为解题习惯。
