### 模块一：舞台（坐标与距离）

想象一个十字路口，横着的是 $x$ 轴，竖着的是 $y$ 轴。
*   **点：** 用 $(x, y)$ 表示。比如 $(3, 4)$ 就是向右走3步，向上走4步。

**死记硬背公式1：两点之间的距离**
如果你有两个点 $A(x_1, y_1)$ 和 $B(x_2, y_2)$，想知道它们离多远？
*   **口语版：** 横坐标差的平方 + 纵坐标差的平方，然后开根号。
*   **数学版：** $\sqrt{(x_1-x_2)^2 + (y_1-y_2)^2}$
*   *例子：* $(0,0)$ 到 $(3,4)$ 的距离？ $\sqrt{3^2 + 4^2} = \sqrt{9+16} = \sqrt{25} = 5$。
    *(这个勾股数 3-4-5 必须背下来，考场上经常出)*

### 模块二：剑（直线）

直线在考试里只有两种长相，你要学会“看脸识人”。

**长相 A：斜截式（最好用的样子）**
$$y = kx + b$$
*   **$k$ (斜率)：** 代表直线的**陡峭程度**。
    *   $k > 0$：上坡（撇）。
    *   $k < 0$：下坡（捺）。
*   **$b$ (截距)：** 代表直线与 $y$ 轴在哪里相交。

**长相 B：一般式（考卷上的样子）**
$$Ax + By + C = 0$$
*   考卷上通常给这个样子（比如 $2x - y + 3 = 0$）。
*   **怎么办？** 把它从长相B“整容”成长相A。
    *   *移项大法：* 把 $y$ 留在一边，其他的扔到对面。
    *   $2x + 3 = y$ $\rightarrow$ $y = 2x + 3$。现在你知道 $k=2, b=3$ 了。

**死记硬背公式2：点到直线的距离（五星级核心）**
这是整个解析几何里**最难记但最重要**的公式。
有一个点 $(x_0, y_0)$，有一条线 $Ax + By + C = 0$。求距离 $d$。

$$d = \frac{|Ax_0 + By_0 + C|}{\sqrt{A^2 + B^2}}$$

*   **人话翻译：**
    *   **分子：** 把点的坐标直接塞进直线方程里，取绝对值（保证是正数）。
    *   **分母：** 直线 $x$ 和 $y$ 前面系数的平方和，开根号。

### 模块三：盾（圆）

圆在考试里也只有两种长相。

**长相 A：标准方程（一眼能看穿的样子）**
$$(x-a)^2 + (y-b)^2 = r^2$$
*   **圆心：** 就是括号里的数变号 $\rightarrow$ $(a, b)$
*   **半径：** 等号右边的数开根号 $\rightarrow$ $r$
*   *例子：* $(x-1)^2 + (y+2)^2 = 9$
    *   圆心是 $(1, -2)$，半径是 $3$（不是9！）。

**长相 B：一般方程（要把你绕晕的样子）**
$$x^2 + y^2 + Dx + Ey + F = 0$$
*   这是把上面那个公式展开后的样子。
*   **怎么办？** 必须学会**“配方法”**把它还原回去。（这是你唯一的难点，下面有教程）。

### 模块四：必杀技（配方法）

这是你从“忘了基础”到“能做题”必须跨过的唯一门槛。
当你看到 $x^2 + y^2 - 4x + 6y = 12$ 这种东西，怎么找圆心半径？

**操作步骤（拿笔练一遍）：**
1.  **分家：** 把 $x$ 的放一起，把 $y$ 的放一起。
    $(x^2 - 4x) + (y^2 + 6y) = 12$
2.  **找伴侣（关键）：** 看 $x$ 一次项系数（这里是 -4），除以2，再平方。$(-4 \div 2)^2 = 4$。
    看 $y$ 一次项系数（这里是 6），除以2，再平方。$(6 \div 2)^2 = 9$。
3.  **两边加：** 方程两边同时加上这两个数。
    $(x^2 - 4x \mathbf{+ 4}) + (y^2 + 6y \mathbf{+ 9}) = 12 \mathbf{+ 4 + 9}$
4.  **折叠：**
    $(x-2)^2 + (y+3)^2 = 25$
5.  **结论：** 圆心 $(2, -3)$，半径 $5$。
