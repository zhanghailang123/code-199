## 核心内容

### 1. 条件推理

**基本形式**:
- 如果 P，那么 Q（P → Q）
- P 是 Q 的**充分条件**
- Q 是 P 的**必要条件**

**四种命题关系**:

| 命题 | 形式 | 关系 |
|------|------|------|
| 原命题 | P → Q | - |
| 逆命题 | Q → P | 与原命题无必然联系 |
| 否命题 | ¬P → ¬Q | 与原命题无必然联系 |
| 逆否命题 | ¬Q → ¬P | 与原命题等价 |

### 2. 逻辑联结词

- **且（∧）**: 两者都满足
- **或（∨）**: 至少一个满足
- **非（¬）**: 否定
