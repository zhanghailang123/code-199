## 题目
已知圆、正方形、等边三角形的周长分别为 a, b, c. 若它们的面积相等，则（）。

## 选项
A. a < b < c
B. a < c < b
C. b < a < c
D. b < c < a
E. c < b < a

## 答案
A

## 解析
这是一个典型的等周问题。该问题探讨的是在周长固定的情况下，哪种图形的面积最大。结论是，在所有周长相同的平面图形中，圆的面积最大。 反之，如果面积相等，则边数越多的正多边形周长越小，圆的周长最小。

**具体计算步骤如下：**
1.  **设定面积**：设三者的面积均为 A。
2.  **圆**：
    *   面积 A = πr²  => r = √(A/π)
    *   周长 a = 2πr = 2π√(A/π) = 2√(πA)
    *   a² = 4πA ≈ 12.57A
3.  **正方形**：
    *   面积 A = s² => s = √A
    *   周长 b = 4s = 4√A
    *   b² = 16A
4.  **等边三角形**：
    *   面积 A = (√3/4)x² => x = √(4A/√3)
    *   周长 c = 3x = 3√(4A/√3) = √(36A/√3) = √(12√3 * A)
    *   c² = 12√3 * A ≈ 20.78A
5.  **比较大小**：
    *   比较 a², b², c² 的系数：12.57 < 16 < 20.78。
    *   因此，a² < b² < c²。
    *   由于 a, b, c 均为正数，所以 a < b < c。
